METAMASK CONNECTION BUTTON

JAVASCRIPT:

var web3; async function Connect(){ await window.web3.currentProvider.enable(); web3=new web3(window.web3.currentProvider); }

CSS:

add position:absolute; top:0; right:0 button { position:absolute; top:0; right:0 background-color: #ffda09; color: #010508; text-align: center; font-size: 16px; padding: 1px; width: 100px; height: 30px; -webkit-transition: all 0.5s; -moz-transition: all 0.5s; -o-transition: all 0.5s; transition: all 0.5s; cursor: pointer; }
This was a fun easy build. To expand on this, one could try to change the position of the button along with styling etc. There are a few different ways of implementing a wallet connect button and this is just one of many using one of the simplest methods.
